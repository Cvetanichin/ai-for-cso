# M1: AI Foundations for CSO Work

**Course module 1 of 7 — "AI at Work for Civil Society"**

## Learning Objectives

By the end of this module, participants will be able to:
1. Explain what AI is — and isn't — in plain language relevant to NGO work.
2. Identify at least three CSO workflows where AI can save measurable time.
3. Apply the four-step prompt formula (Role, Task, Context, Constraints) to any CSO task.
4. Run a basic AI risk assessment before adopting a new tool.
