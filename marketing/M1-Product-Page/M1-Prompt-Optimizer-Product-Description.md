## How to Use

**Local use:**
```bash
pip install -r requirements.txt
python prompt_optimizer.py --input "Draft a report" --task "donor_reporting"
```

**Deploy the web app:** follow the deployment guide in the README, then share the link with your team.

**Integrate with WordPress:** call the `/api/optimize` endpoint from a custom plugin or block.
