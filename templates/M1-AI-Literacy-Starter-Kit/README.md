# AI Literacy for NGOs: Starter Kit

**The fastest way for CSO professionals to start using AI — no technical skills required.**

## What's Included

- **AI Prompt Design Cheat Sheet** (`files/AI-Prompt-Design-Cheat-Sheet.md`) — the four-step prompt formula, plus five ready-to-adapt prompts for grant writing, donor reporting, M&E, stakeholder comms, and fundraising research.
- **AI Risk Assessment Checklist** (`files/AI-Risk-Assessment-Checklist.md`) — nine questions to run before adopting any AI tool, covering data privacy, bias, over-reliance, and ethics.
- **AI Tool Selection Guide** (`files/AI-Tool-Selection-Guide.md`) — ten free/low-cost AI tools relevant to CSO work, with honest pros and cons.
- **Notion Prompt Library Template** (`notion-templates/Prompt-Library-Template.csv`) — importable starter table for your team's shared prompt library.
- **Prompt Optimizer Web App** — free access (see `/ai-tools/M1-Prompt-Optimizer/`), no setup required for buyers.

## Core Transformation

Buyers go from "I don't know where to start with AI" to having a working prompt formula, a safe-adoption checklist, and a live tool they can use the same day — without writing a line of code.

## How to Use

1. Start with the **AI Tool Selection Guide** — pick one or two tools to test.
2. Use the **Prompt Design Cheat Sheet** to write your first prompts.
3. Run any new tool through the **Risk Assessment Checklist** before relying on it.
4. Try the **Prompt Optimizer web app** to apply the four-step formula automatically.
5. Save what works in your **Notion Prompt Library**.
