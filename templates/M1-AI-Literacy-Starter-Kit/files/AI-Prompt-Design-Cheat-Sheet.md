# AI Prompt Design Cheat Sheet for CSO Professionals

*Part of the "AI Literacy for NGOs: Starter Kit" — Cvetanichin Consultancy*

## The Four-Step Prompt Formula

| Step | What It Does | Ask Yourself |
|------|---------------|---------------|
| **1. Role** | Sets the AI's expertise and tone | "Who should the AI act as?" |
| **2. Task** | Defines the specific output | "What exactly do I need produced?" |
| **3. Context** | Gives background the AI needs | "What does it need to know to get this right?" |
| **4. Constraints** | Sets format, length, tone limits | "What rules should the output follow?" |

## Example Prompts for CSO Workflows

### Grant Writing
> You are a grant writer for a [NGO type] NGO working in [region]. Draft a [section, e.g. problem statement] for a [amount]€ project on [topic]. Use simple language, avoid jargon, and align with [donor]'s priorities. Keep it under [word limit] words.

### Donor Reporting
> You are a project manager for a [NGO type] NGO. Summarize [number] key achievements from our [time period] report for [donor]. Highlight impact on [target group]. Tone: professional. Length: [word limit] words.

### M&E Analysis
> You are an M&E specialist for a [NGO type] NGO. Analyze this [data type] and identify [number] trends, [number] gaps, and [number] recommendations. Use bullet points and plain language.

### Stakeholder Communications
> You are a communications officer for a [NGO type] NGO. Draft a [comm type — email, social post] to [audience] about [topic]. Tone: [tone]. Length: [word limit] words.

### Fundraising Research
> You are a fundraising manager for a [NGO type] NGO. Research [number] potential donors for a [topic] project in [region]. Focus on donors with [criteria]. Output as a table.

## Common Mistakes to Avoid

| Mistake | Fix |
|---------|-----|
| Too vague ("Write a report.") | Add specificity — what, for whom, why |
| No context ("Summarize this data.") | State what the data is and where it's from |
| Overly complex (multiple goals at once) | One goal per prompt |
| Missing constraints (no length/tone/format) | Always specify format, tone, length |
| Ignoring the audience | State who the output is for |

## Pro Tips

- Save prompts that work well — build a team prompt library (Notion template included in this kit).
- Refine in rounds: if output is too generic, add more context; if too long, add a word limit.
- Never paste personally identifiable beneficiary data into a general-purpose AI tool — see the AI Risk Assessment Checklist in this kit before you do.
